# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog, and this project uses semantic versioning for public releases.

## [v1.0.0] - 2026-05-17

### Added

- Reproducible Python pipeline for responsible credit-growth Next-Best-Action analytics.
- Synthetic financial-services data generation.
- Customer-month feature matrix construction.
- Conversion propensity and responsible credit-behaviour models.
- Expected-value scoring and advisor-ready customer ranking.
- Governance checks for duplicate keys, feature coverage, leakage-like feature names, high missingness, and Top 50 eligibility.
- Streamlit dashboard with main dashboard, executive overview, customer ranking, model performance, and governance monitoring pages.
- Dashboard screenshots embedded in the README.
- Model card, technical note, executive summary, visualisation layer documentation, reproducibility guide, and ML model specification.
- GitHub Actions workflow for automated pipeline and test validation.
- MIT licence.
- Project disclaimer.
- Citation metadata.
- Makefile shortcuts for common reviewer commands.

### Notes

- All data is synthetic.
- The repository is intended for demonstration and decision-support design, not automated credit approval.
